# האתר לשיתוף סרטונים

האתר שומר את קבצי הווידאו המקוריים בתיקיית `uploads` בלי להמיר ובלי לדחוס, כדי שהאיכות לא תרד.

## הפעלה במחשב

```bash
npm start
```

פותחים בדפדפן:

```text
http://localhost:5177
```

## שימוש מטלפון או מחשב אחר באותו Wi-Fi

במחשב שמריץ את האתר מפעילים:

```bash
ipconfig getifaddr en0
```

ואז פותחים במכשיר השני:

```text
http://YOUR-IP:5177
```

כדי לשלוח את האתר למישהי מחוץ לבית צריך להעלות אותו לשרת עם אחסון קבצים, או להשתמש בשירות מנהרה זמני כמו Tailscale / Cloudflare Tunnel / ngrok. האתר עצמו כבר מוכן לזה כי הוא לא דוחס את הקבצים.

## קישור קבוע באינטרנט

כדי שהקישור יהיה פעיל כל הזמן, צריך לפרוס את האתר לשירות שרץ 24/7 עם אחסון קבוע.

הפרויקט כולל קובץ `render.yaml` שמוכן לפריסה ב-Render עם דיסק קבוע בנתיב `/var/data`, כדי שהסרטונים והקובץ `videos.json` יישמרו אחרי הפעלה מחדש.

בהגדרות הדיסק כדאי לבחור נפח שמתאים לסרטונים באיכות המקורית. סרטונים באיכות גבוהה יכולים לתפוס הרבה מקום.
