# פריסה לקישור קבוע

המסלול המוכן בפרויקט הוא Render עם דיסק קבוע. זה מתאים יותר משרת סטטי רגיל כי האתר צריך לקבל העלאות ולשמור קבצי וידאו מקוריים.

## מה צריך

- חשבון GitHub
- חשבון Render
- תוכנית Render עם Persistent Disk

## שלבים

1. מעלים את תיקיית `family-video-share` ל-GitHub.
2. ב-Render בוחרים New Blueprint ומחברים את ה-repository.
3. Render יקרא את `render.yaml`.
4. מגדירים ערך סודי ל-`SHARE_PASSWORD`.
5. מוודאים שקיים דיסק בשם `videos` שמחובר ל-`/var/data`.
6. פותחים את הקישור ש-Render נותן ושולחים לאחותך.

## איכות וידאו

האתר לא דוחס ולא ממיר את הסרטונים. הוא שומר את הקובץ המקורי ומציג אותו ישירות. כפתור "הורדת מקור" מוריד את אותו קובץ שהועלה.

## נפח

ברירת המחדל בפריסה היא דיסק של 20GB ומגבלת העלאה של 20GB לקובץ. אם מעלים הרבה סרטונים באיכות גבוהה, צריך להגדיל את הדיסק ב-Render.
